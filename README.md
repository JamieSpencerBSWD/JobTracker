# Job Application Management System

A full-stack job tracking and management platform designed to organize, analyze, and optimize the job search process.

## Overview

This project is a structured job application management system that replaces spreadsheets with a centralized database-driven workflow.

It allows users to track job applications, categorize them using flexible tagging, and monitor their progress through different stages of the hiring process.

The system is designed as a foundation for future automation, analytics, and job search optimization features.

## Features

- Track job applications across multiple stages (applied, interview, offer, rejected, etc.)
- Full CRUD functionality for applications
- Tagging system for classification:
  - Tech stack (Vue, Node, PostgreSQL, Express)
  - Role type (Frontend, Backend, Fullstack, DevOps, etc.)
  - Source (LinkedIn, Indeed, referral, company site, etc.)
  - Priority levels (dream, high, medium, backup, etc.)
- Advanced filtering and querying of applications
- Structured pipeline for application tracking
- RESTful API for backend operations

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- REST API architecture

## System Design

- Applications are the core entity
